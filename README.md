# SLM Finetuning Dashboard

> A comprehensive training, inference, and evaluation workspace for Small Language Models

## Features

- **Training** - Fine-tune small language models with ease
- **Inference** - Deploy and test models in real-time
- **Evaluation** - Comprehensive metrics and performance analysis
- **Dashboard** - Interactive UI for monitoring and management

## Installation

```bash
# Clone the repository
git clone https://github.com/Rithanya654/SLM-Finetuning-Dashboard.git
cd SLM-Finetuning-Dashboard

# Install dependencies
npm install
pip install -r requirements.txt
```

## Quick Start

```bash
# Start the development server
npm start

# Run the backend
python app.py
```

## Usage

1. **Upload your dataset** - Prepare your training data
2. **Configure training** - Set hyperparameters and model settings
3. **Train your model** - Monitor progress in real-time
4. **Run inference** - Test your fine-tuned model
5. **Evaluate results** - Analyze performance metrics
